UK Weather Data API Dashboard

Django REST API for UK MetOffice Weather Data

API Endpoints

Regions: /api/regions/
Get all UK regions (England, Scotland, Wales, etc.)
Weather Parameters: /api/parameters/
Get weather measurement types (Temperature, Rainfall, etc.)
Weather Data: /api/weather-data/
Get historical weather measurements

API Filtering Examples:

Filter by region:
/api/weather-data/?region__code=UK

Filter by parameter:
/api/weather-data/?parameter__code=Tmax

Filter by year:
/api/weather-data/?year__gte=2020
